# 🐘 Simulation de Zoo en Python

## Description
Ce projet est une application de simulation de zoo développée en Python.  
Il permet de gérer des animaux, de simuler leur évolution dans le temps et de sauvegarder leurs données.

Le projet utilise la programmation orientée objet (POO) et une base de données SQLite.

---

##  Objectifs du projet
- Appliquer la programmation orientée objet (POO)
- Manipuler une base de données
- Simuler un système évolutif (temps réel simplifié)
- Organiser un projet propre et structuré

---

##  Architecture du projet

Le projet est organisé en 3 parties :

### 🐾 Animal
Représente un animal avec :
- nom
- espèce
- faim
- énergie

###  Zoo
Gère les actions principales :
- ajouter un animal
- afficher les animaux
- nourrir un animal
- lancer la simulation

### Database
Gère le stockage des données avec SQLite :
- création de la base
- insertion des animaux
- mise à jour des données
- récupération des informations

---

## Fonctionnalités

- Ajouter des animaux
- Afficher la liste des animaux
- Nourrir un animal
- Simulation du temps (faim augmente, énergie diminue)
- Sauvegarde automatique des données

---

##  Technologies utilisées

- Python
- SQLite (base de données)
- Programmation orientée objet (POO)
