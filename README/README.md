# Simulation de Zoo

Ce projet est une application en Python qui simule un zoo.

Fonctionnalités :
- Ajouter des animaux
- Voir les animaux
- Nourrir les animaux
- Simulation du temps (faim et énergie)

Technologie utilisée :
- SQLite (base de données)

Lancer le projet :
python zoo.py
